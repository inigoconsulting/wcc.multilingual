wcc.multilingual Installation
-----------------------------

To install wcc.multilingual using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.multilingual`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.multilingual

* Re-run buildout, e.g. with:

    $ ./bin/buildout
